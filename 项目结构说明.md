## 数据流转图

### 用户访问流程

```
用户浏览器
    ↓
前端路由（Vue Router）
    ↓
页面组件（.vue文件）
    ↓
API模块（api/*.js）
    ↓
HTTP请求（Axios）
    ↓ 
后端控制器（Controller）
    ↓
业务服务层（Service）
    ↓
数据访问层（Mapper）
    ↓
MySQL数据库
```

### 认证授权流程

```
用户登录
    ↓
AuthController.login()
    ↓
UserService.login()
    ↓
验证用户名密码
    ↓
生成 JWT Token
    ↓
返回 Token 给前端
    ↓
前端存储 Token
    ↓
后续请求携带 Token
    ↓
JwtInterceptor 拦截验证
    ↓
通过后访问资源
```

## 技术选型说明

### 为什么选择这些技术？

#### Spring Boot 3.x
- ✅ 简化配置，开箱即用
- ✅ 生态完善，社区活跃
- ✅ 性能优秀，稳定可靠

#### Vue 3
- ✅ 组合式API，代码更清晰
- ✅ 性能提升，响应更快
- ✅ TypeScript支持更好

#### MyBatis Plus
- ✅ 简化CRUD操作
- ✅ 代码生成器方便
- ✅ 分页插件强大

#### Element Plus
- ✅ 组件丰富完整
- ✅ 支持Vue 3
- ✅ 可定制主题

#### JWT
- ✅ 无状态认证
- ✅ 跨域支持好
- ✅ 移动端友好

## 扩展开发指南

### 如何添加新功能？

#### 1. 添加新的实体（例如：友情链接）

**后端步骤**:
```
1. 创建数据表 (SQL)
2. 创建实体类 (entity/FriendLink.java)
3. 创建Mapper (mapper/FriendLinkMapper.java)
4. 创建Service (service/FriendLinkService.java)
5. 创建ServiceImpl (service/impl/FriendLinkServiceImpl.java)
6. 创建Controller (controller/FriendLinkController.java)
```

**前端步骤**:
```
1. 创建API模块 (api/friendlink.js)
2. 创建页面组件 (views/guest/FriendLink.vue)
3. 添加路由 (router/index.js)
4. 添加导航菜单
```

#### 2. 添加新的管理页面

```
1. 创建管理端接口 (controller/admin/AdminXXXController.java)
2. 创建管理端页面 (views/admin/XXX.vue)
3. 在 AdminLayout.vue 添加菜单
4. 在 router/index.js 添加路由
```
