- CPU: 2核心以上

### 软件环境
- [x] JDK 17+
- [x] MySQL 8.0+
- [x] Node.js 16+
- [x] Nginx 1.18+

### 部署步骤
1. 导入数据库
2. 配置后端（修改application.yml）
3. 启动后端服务
4. 构建前端（npm run build）
5. 配置Nginx
6. 启动服务

## 常见问题解答

**Q: 如何修改管理员密码？**
A: 登录后台后，可以通过用户管理或直接修改数据库。

**Q: 如何自定义主题颜色？**
A: 修改 `frontend/src/styles/harmony.scss` 中的颜色变量。

**Q: 如何备份数据？**
A: 定期导出MySQL数据库，备份上传文件目录。

**Q: 支持多用户吗？**
A: 支持，普通用户可以注册、登录、评论、留言。

**Q: 如何启用HTTPS？**
A: 使用 Let's Encrypt 申请免费证书，配置Nginx。
