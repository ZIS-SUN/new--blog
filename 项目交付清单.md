## 🚀 十三、使用指南

### 13.1 快速开始（3步）

```bash
# 步骤1: 导入数据库
mysql -u root -p < database/blog_schema.sql
mysql -u root -p < database/test_data.sql

# 步骤2: 启动后端（新终端）
cd backend
mvn spring-boot:run

# 步骤3: 启动前端（新终端）
cd frontend
npm install
npm run dev
```

### 13.2 测试系统（3步）

```bash
# 步骤1: 确保后端已启动

# 步骤2: 运行自动化测试
chmod +x test_api.sh
./test_api.sh

# 步骤3: 访问系统
# 前端: http://localhost:3000
# 后端API: http://localhost:8080/api
# Swagger: http://localhost:8080/api/swagger-ui.html
```
