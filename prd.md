# 基于 Spring Boot 的个人博客系统 - PRD 文档（增强版）

## 一、项目概述

### 1.1 项目背景

独立内容创作者与技术博主需要一个统一的平台，兼具写作、展示、互动与运营功能。现有平台受限于模板与广告，难以个性化。
本系统旨在提供一个可私有部署的个人博客系统，支持公告发布、访客互动与数据可视化，采用鸿蒙设计语言以提升现代感与一致性。

### 1.2 项目目标

* 实现 Markdown 写作与文章管理
* 增加公告发布与展示功能
* 提供访客评论、留言互动
* 支持文章分类、标签、归档与搜索
* 实现后台管理（用户、文章、公告、评论）
* 采用鸿蒙设计体系统一视觉体验
* 提供访问数据统计与图表分析

### 1.3 技术选型

* **后端框架**：Spring Boot 3.x
* **前端框架**：Vue 3.x + Element Plus（自定义鸿蒙风格主题）
* **UI 设计规范**：Harmony Design Language（圆角、自然动效、流动布局）
* **安全认证**：JWT + MD5
* **数据库**：MySQL 8.0
* **编辑器**：Editor.md
* **部署环境**：Nginx + Docker（可选）
* **可视化**：ECharts

### 3.2 模块说明

#### 3.2.1 首页

* 卡片式布局（鸿蒙圆角 + 弹性阴影）
* Banner 展示最新公告或置顶文章
* 列表分页加载文章摘要
* 侧边栏：热门标签、分类、作者简介、公告入口

#### 3.2.2 公告中心（新增）

* 公告列表页：标题、发布日期、摘要
* 公告详情页：完整内容、作者、发布时间
* 支持“置顶公告”与“紧急公告”标识（红色标签）
* 公告自动轮播展示在首页 Banner 区

#### 3.2.3 文章详情页

* Markdown 渲染
* 点赞、阅读数统计
* 评论区嵌套展示，带楼层结构
* 自动目录 TOC
* 动效：平滑滚动与淡入展示

#### 3.2.4 分类 / 标签 / 归档

* 分类与标签独立页面
* 时间轴归档视图（年/月）
* 动态路由切换

#### 3.2.5 搜索功能

* 支持标题、正文模糊搜索
* 热词联想与历史搜索记录

#### 3.2.6 评论互动

* 登录用户可发表评论与回复
* 未登录用户提示注册登录
* 管理员可后台审核

#### 3.2.7 关于我

* 展示头像、简介、技能、联系方式、社交链接
* 使用鸿蒙“信息卡片”布局

#### 3.2.8 留言板

* 留言发布与回复
* 展示留言时间轴

#### 3.2.9 登录注册 / 退出

* 登录返回 JWT Token
* 注册验证邮箱唯一性
* 支持邮箱重置密码

### 4.2 模块说明

#### 4.2.1 仪表盘首页

* 数据卡片：文章数、评论数、访问量、公告数
* 折线图：日访问量趋势
* 饼图：文章分类占比
* 最新留言与评论快速入口

#### 4.2.2 文章管理

* Markdown 编辑器，支持上传封面图
* 草稿箱、已发布、置顶状态管理
* 预览、批量删除、搜索过滤

#### 4.2.3 分类 / 标签管理

* 添加、修改、删除分类与标签
* 支持统计每类文章数量

#### 4.2.4 公告管理（新增）

* 列表显示公告标题、发布时间、状态
* 创建公告：标题、内容、是否置顶、是否紧急
* 富文本编辑器（支持图片插入）
* 置顶公告前台优先展示
* 删除公告需二次确认

#### 4.2.5 评论与留言管理

* 支持状态筛选（待审 / 已审 / 屏蔽）
* 支持批量审核与删除
* 评论内容预览

#### 4.2.6 用户管理

* 用户信息管理：注册时间、邮箱、状态
* 支持禁用 / 启用账户

#### 4.2.7 系统设置

* 博客名称、副标题、作者信息
* 社交媒体配置
* Token 过期时间、上传文件大小限制

#### 4.2.8 数据统计

* 访问量统计、评论数趋势、热门文章榜单
* 每日新增用户与留言数量
* 可导出 Excel 报表

#### 4.2.9 日志管理

* 登录日志、操作日志、系统异常
* 时间范围查询与导出

## 六、UI 设计规范（鸿蒙风格）

* **主色调**：#007DFF（天青蓝）
* **次色调**：#E6F0FF（柔和蓝白背景）
* **字体**：HarmonyOS Sans
* **圆角**：统一 12px
* **按钮与卡片动画**：采用渐变与弹性过渡
* **图标库**：Harmony Icons
* **整体风格关键词**：清爽、流动、亲和、轻动效

---

## 七、安全与权限

* 密码 MD5 加密
* JWT 鉴权
* 前后端 Token 验证
* 后台权限控制：管理员 / 普通用户
* 公告发布仅限管理员

* 图片大小 ≤ 2MB
