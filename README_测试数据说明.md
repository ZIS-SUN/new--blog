## 🔧 如何导入测试数据

### 方法一：使用命令行

```bash
# 1. 导入基础表结构
mysql -u root -p < database/blog_schema.sql

# 2. 导入测试数据
mysql -u root -p < database/test_data.sql
```

### 方法二：使用 MySQL Workbench

1. 打开 MySQL Workbench
2. 连接到数据库
3. 选择 File > Run SQL Script
4. 先导入 `blog_schema.sql`
5. 再导入 `test_data.sql`

### 方法三：使用 Navicat

1. 打开 Navicat
2. 连接到 MySQL
3. 右键数据库 > Execute SQL File
4. 依次选择两个 SQL 文件执行
